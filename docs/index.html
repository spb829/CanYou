Technical Features

Infinite paging

메인화면에 보여지는 스크롤뷰는 한쪽으로 무한히 돌아갈 수 있도록 커스터마이징 되었습니다.
스크롤시마다 UIScrollView의 Offset을 재조정하여 서브뷰들을 재배치하여 구현하였습니다.

UIKit, UIScrollView, UIPageControl

RealmSwift

데이터 관리에는 Realm을 사용하였습니다.
iOS뿐만아니라 안드로이드도 지원하여 많이 사랑받는 데이터베이스 시스템이라 선택하게 되었습니다.

Core Animation

시작시 로딩화면에서 Label의 텍스트를 바꿀때와
네이베이션 바를 유지하고 싶지만 modally하게 설정 뷰를 표시하고 싶어
커스텀 애니매이션을 만들어 사용하게 되었습니다.

Watch Connectivity

생소하고 배우지 않았던 것이라 애플워치 연동은 하나의 도전이었습니다.
AppDelegate를 활용하여 WCSession을 활성화하고 백그라운드에서도 통신할 수 있게 구현하였습니다.
하지만 Realm이 thread-safe한 데이터베이스라 다소 어려운 점도 있었습니다.

WatchKit, WatchConnectivity, WCSession
